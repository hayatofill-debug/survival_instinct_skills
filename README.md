# Survival_instinct_skills

このMODは、**Pufferfish’s Skills** のカスタムアドオン（スキルツリー追加Mod）です。
公式のスキルを分かりやすくまとめ、さらに独自のスキルを複数追加して、より奥深いスキルシステムを楽しめるようにしています。

## ⚠️ 前提MOD (Requirements)
このMODを動作させるには、以下のMODが必須です。必ず一緒に導入してください。
* **[Pufferfish’s Skills](https://www.curseforge.com/minecraft/mc-mods/puffish-skills)**
* **[Pufferfish's Attributes](https://www.curseforge.com/minecraft/mc-mods/puffish-attributes)**

## 📥 ダウンロード (Downloads)
公式の配布先は以下のリンクのみとなります。
* [CurseForge](https://www.curseforge.com/minecraft/mc-mods/survival-instinct-skills)
* [Modrinth](https://modrinth.com/project/apNcy79D)

## ⚙️ MODの仕様 (Specifications)
* **対応ローダー：** NeoForge / Fabric
* **対応バージョン：** 1.21.3 - 1.21.x (26.1 - 26.x)
* **対応言語：** 日本語と英語

### 概要
[Pufferfish's Template Generator](https://puffish.net/skillsmod/editor/) を用いて作成したデータをベースに、[公式Wiki1 (Skills)](https://puffish.net/skillsmod/docs/) と [公式Wiki2 (Attributes)](https://puffish.net/attributesmod/docs/) に記載されているほぼすべてのスキルを日本語化してまとめ、さらに独自のスキルをいくつか書き加えたものです。

## 🛠️ 導入方法 (Installation)
1. 配布先（CurseForge / Modrinth）から `.jar` ファイルをダウンロードします。
2. Minecraftの `mods` フォルダにそのまま入れてください。
*(※GitHubからソースを利用する場合は、`LICENSE` と `README.md` を除いたファイル群を `.jar` 形式にビルド・パックしてご使用ください。)*
